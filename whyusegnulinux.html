Why use GNU/Linux

Posted on 7 March by Tomas Mozdren
  1. Linux itself
    Linux can be run on everything, well mostly everything. You can run it on singleboard computers like the Raspberry Pi or it is used on supercomputers.
  2. Installing apps is easier and safer
    Unlike Windows where you have to get your apps from some shady website, Linux uses something called a package manager. For debian based distros its apt. All they have to do is run a simple command and the app is installed.
  3. Updating is easier and faster too
    Like metioned in point 2 you can install apps using a package manager, but you can also update apps on your system with one command and the same applys to system updates. You can even update while you are using your computer, thats thanks to Linux using something called the Live kernel. There will be some updates that will need to ask you to restart the system, but that happends rarely.
  4. Linux is more secure
    With Linux being open source that does not mean that it is a bad thing, in fact its better because you have a large amount of people looking at the code, so it is much hard for malware and viruses to make its way into your system. That does not mean that you should try to make viruses for it thou.
  5. Can revive an old/slow computer
    With Linux being lighweight and having so much overhead, it makes it the ideal way to increase the lifespan of an old computer or laptop.
  6. Can be learned quickly
    Yes you can learn Linux quickly thanks to its documentation. If you do not like reading you can always try using Linux. Either in a virtual machine or on real hardware. There are also forums and groups of people that make sure to help those in need. If you have any questions, do not be afraid to ask.
  7. Ideal enviroment for work
    So yes you can do some real work on Linux. From simple document editing to complex tasks like coding or even AI.
